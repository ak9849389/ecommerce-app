# E-commerce App

## Features
- User registration and login
- Product listing and details
- Shopping cart
- Checkout and order history

## Getting Started

1. Copy `.env.example` to `.env` and set your values.
2. Run `npm install`.
3. Start the server: `npm run dev`.
4. Open your browser at `http://localhost:3000`.

## Technologies Used
- Node.js
- Express.js
- MongoDB
